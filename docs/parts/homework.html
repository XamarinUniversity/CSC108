Homework goals

Here you will write a console number guessing game. Sample output from the completed application looks like this, notice as you make guesses the amount of available tries decreases.

You will need to perform the following steps:

  1. Build a GuessingGame class to manage generating a random number and making a guess. This includes:
    • Two public constant fields to store the upper and lower boundaries for the random number.
    • A public read only auto property to store the actual random number.
    • A public read only auto property to store the amount of tries remaining.
    • A constructor that sets the starting amount of tries and generates the random number.

      • Hint: To generate a random number use the Random class. The Random class has a method called Next that takes in a lower and upper boundary. Note that the upper boundary is not inclusive.

    • A public method that returns a bool to make a guess.

      • Hint: If the user guesses a number outside the boundaries, report this by throwing an ArgumentOutOfRangeException.

  2. Create a GuessingGame variable in the Main method.
  3. Instantiate your GuessingGame variable.
  4. Create a game loop that continues to run as long as the game is not over.
  5. For every iteration ask the user to enter a number.
  6. Invoke the appropriate methods and properties on your GuessingGame instance. If the user is out of tries, don't allow them to make any guesses.

    7. Hint: Make sure you catch the ArgumentOutOfRangeException here. If you don't, your program will crash.

  7. If the user found the hidden number, display a winner message.

Solution
